# GB Hello World

Simple hello world from the [gb asm tutorial].

## Dependencies

* [rgbds]
* [bgb]

## Compile

`make`

## Run

`bgb -rom bin/hello-world.gb`


[gb asm tutorial]:  https://eldred.fr/gb-asm-tutorial/hello-world.html
[rgbds]:            https://github.com/gbdev/rgbds
[bgb]:              https://bgb.bircd.org/
